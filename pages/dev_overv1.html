Select Project: CourtSMS MVP

A simple text messaging tool that pulls the latest court date information and feeds it into a database. You text in with the system, providing your first and last name. If a corresponding court date is found from this data, then the system will text back and inform the user of said date and location. The tool is fully functional and conceptually based on CourtBot, a previous Code for America project. Furthermore, it is presently pulling from a valid and up to date secure FTP with the latest court date. Indeed, this project could be rolled out at any moment to the public. So why donâ€™t we do it? Our team believes that rolling this tool out first would in fact limit the ability for ClientComm to become fully fleshed. As a result, we are rolling it out, but as a feature within ClientComm. The purpose for this is that it enables us to focus more deeply on user needs in creating a successful messaging and case management communications tool that facilitates longitudinal communication and support.
