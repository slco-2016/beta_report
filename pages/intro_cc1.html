How ClientComm Works

Every conversation has a subject and each contact from a client has a title for organizational purposes. From the conversation screen, you can send and receive messages with your client. With ClientComm, you can change the contact method from within the same conversation. This allows a single conversation thread to occur across multiple devices.

Not only is there an organizational benefit to unique conversations, but it allows multiple case managers to work with the same client, with the case manager currently assisting the client engaged in the open conversation. This is particularly beneficial for environments such as drug court, where therapists, case managers, and others might all be working with the same client concurrently.
