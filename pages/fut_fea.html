Future Features

Looking Ahead

In addition to improved metrics, there are a variety of in-demand features that case managers have requested numerous times. A highlight of some features that are currently in development:

  • Templating tool for repeat or common messages.
  • Sending messages to multiple clients at the same time.
  • Timed notifications for automated message sending for case managers.

In addition to these soon-to-be-released features, some more broadly desired features that have been desired by users include:

  • Email integration
  • Voicemail transcription
  • Grouping clients
  • Tagging and codification management tools
  • Client-facing components (including a checklist)
